# Q & A
Q：这个工作流的作用是什么？

A：放一张参考图，放一张产品图，自动帮你合成。你也可以忽略IPadapter节点，使用文字描述生成背景。



Q：怎么用这个工作流？

A：放一张背景参考图（如果有参考图有物体可以用蒙版擦除），放一张产品图，点击添加提示词队列，就完成了。
